# Mark Oduor | Systems Architect & Digital Engineer Portfolio

A premium, high-performance portfolio website designed for a modern Systems Architect. This project features a clean "Silicon Valley Corporate" aesthetic, interactive filtering, and smooth, 60fps animations.

## 🚀 Live Demo
[Insert your hosted URL here, e.g., https://markoduor.vercel.app]

## 🛠️ Tech Stack
- **Structure:** Semantic HTML5
- **Styling:** CSS3 (Custom Design System with Glassmorphism)
- **Interactions:** Vanilla JavaScript (ES6+)
- **Backend:** Formspree API (Contact Form)
- **Animations:** Intersection Observer API & CSS Keyframes

## ✨ Key Features
- **Filterable Portfolio:** Sort projects by Web Development, Applications, or Data Systems.
- **Robust Image Fallbacks:** Automatically displays professional icons if project images are missing.
- **Interactive Experience Timeline:** Clickable nodes to explore career history.
- **Conversion Toast:** Intelligent slide-in notification to encourage recruiter engagement.
- **Bento Grid Layout:** Modern, responsive education and about sections.
